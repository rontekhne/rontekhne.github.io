Printf

Nesta aula nós vamos implementar nosso primeiro programa em C usando a função printf. Além disso, vamos descobrir como um programa em C realmente funciona, como é compilado, executado e como podemos depurá-lo quando encontrarmos alguma falha.

Exercício 2.1
  1. Você sabe o que é "ascii art"? Pesquise na internet e implemente um programa que desenhe na tela uma ascii art.
Cabeçalhos

Nesta aula usaremos outras funções de saída como putchar e puts, além de entender melhor como funciona uma biblioteca e o que é um arquivo de cabeçalho. Usaremos as funções sleep e usleep da biblioteca unistd.h para pausar uma thread.

Exercício 2.2
  1. Use a função usleep para criar um programa que simule um karaokê. Escreva a letra de uma música que você goste dentro de uma das funções de saída que você aprendeu. Entre cada função de saída, coloque uma pausa com uma das funções de pausa de thread. Execute o seu programa e cante como se estivesse em um karaokê de Tóquio.
Cores

Nesta aula usaremos as cores ANSI para fazer as nossas saídas ficarem mais bonitas e charmosas. Criaremos um arquivo de cabeçalho para definir as cores e importaremos esse arquivo para dentro do nosso programa.

Exercício 2.3
